# Activity_Insight
Проект Activity Insight представляет собой настольное приложение для анализа пользовательской активности. 
Система отслеживает, какие программы использует пользователь, определяет периоды продуктивной работы и отдыха, 
сохраняет статистику локально и визуализирует её в удобном интерфейсе. 
В отличие от существующих решений, проект делает упор на приватность, офлайн-работу и возможности классификации приложений.
